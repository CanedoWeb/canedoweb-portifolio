site institucional hamgúrgueria grelha urbana

O projeto desenvolvido para a Grelha Urbana é um site institucional moderno e impactante, criado para transmitir toda a personalidade da marca, que leva o hambúrguer a sério. Desenvolvido 100% com HTML, CSS e animações com GSAP, o site oferece uma navegação fluida e visual marcante, combinando elementos visuais robustos com uma paleta de cores forte e tipografia ousada. O objetivo foi proporcionar uma experiência imersiva, digna de uma hamburgueria urbana, artesanal e autêntica.

Página inicial com destaques do mês animados, seção de hambúrgueres organizada por categorias, design responsivo, adaptado para dispositivos móveis, identidade visual forte baseada na temática artesanal, com cores escuras e destaque para o amarelo vibrante, tipografia personalizada que reforça o conceito da marca


Tecnologias utilizadas:

  • Figma para prototipação de design
  • HTML5 e CSS3
  • JavaScript com GSAP para animações
  • Layout responsivo com CSS Grid e Flexbox


*Loja Inspirada na Marca Nike, sem objetivos lucrativos, é apenas uma demonstraçao.
