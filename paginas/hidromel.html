slider - galeria de hidroméis: hidromel skal

Este projeto consiste em uma galeria interativa e responsiva, criada com foco em apresentar diferentes rótulos de hidromel de forma visualmente atrativa. Utilizando HTML, CSS Grid e princípios de design responsivo, a vitrine exibe uma coleção de garrafas personalizadas, com cada imagem representando uma variação do produto. Objetivos do projeto: Apresentar de forma elegante os produtos da marca. Garantir que as imagens se ajustem ao tamanho do container sem distorções. Criar uma experiência visual coesa e adaptável a qualquer tela.

Página inicial com destaques do mês animados, seção de hambúrgueres organizada por categorias, design responsivo, adaptado para dispositivos móveis, identidade visual forte baseada na temática artesanal, com cores escuras e destaque para o amarelo vibrante, tipografia personalizada que reforça o conceito da marca


Tecnologias utilizadas:

  • Figma para prototipação de design
  • Wordpress Elementor
  • JavaScript para animações
  • Layout responsivo com CSS Flexbox
