site institucional multipágina - engelux engenharia

O site institucional desenvolvido para a Engelux Engenharia foi pensado para refletir os principais valores da empresa: solidez, confiança e excelência. O projeto tem como foco destacar a credibilidade da marca no setor da construção civil e engenharia corporativa, oferecendo uma navegação clara, moderna e eficiente. Totalmente construído com HTML5, CSS3 e JavaScript com GSAP, o layout é 100% responsivo, adaptando-se perfeitamente a qualquer tipo de dispositivo. A interface é composta por um design limpo e elegante, utilizando tons corporativos, elementos geométricos, animações leves e uma hierarquia visual bem definida. O site apresenta seções organizadas de forma lógica e intuitiva, como: Sobre a empresa, Portfólio de projetos, Áreas de atuação, Sustentabilidade e Contato


Tecnologias utilizadas:

  • Figma para prototipação de design
  • HTML5 e CSS3
  • JavaScript com GSAP para animações
  • Layout responsivo com CSS Grid e Flexbox
